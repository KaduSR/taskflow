# TaskFlow 📋

> Kanban board colaborativo com notificacoes em tempo real para times remotos.

## Stack
React 18 • Firebase (Auth, Firestore) • TypeScript • react-beautiful-dnd

## Problema
Times remotos sem visibilidade do fluxo de trabalho — tarefas se perdem, 
prazos nao sao cumpridos, nao ha accountability.

## Solucao
Board Kanban com drag & drop, convite de membros e notificacoes Firestore em tempo real.

## Metrica
Zero conflitos de edicao simultanea com Firestore real-time sync.
